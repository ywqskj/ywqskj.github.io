    某某酒店弱电应知应会¶

    1.常见网络故障¶

    1.1 WiFi故障¶

    **集团网关认证地址：**192.168.50.254 1.全区域WiFi可以连接但无网络（网关死机或数据阻塞）； 解决方法：重启服务器机房网关 2.全区域无法连接（由于网关不能分配IP地址，地址池使用完毕）； 解决方法：重启服务器机房网关 3.个别区域WiFi出现连接后无法使用； 解决方法：到相应楼层弱电井检查网络交换机光纤跳线指示灯是否正常，如不正常更换光纤跳线到别的网路交换机或更换光模块后查看是否正常； 4.个别房间AP无网络或无法连接 解决方法：检查弱电井到AP网线是否正常，如不正常修复网线；网线正常AP无法使用，更换别的房间可以使用AP一台查看是否解决；通过AC管理器查看坏的AP是否在线，如不在线对AP进行配置； 5.手机或者电脑无法自动获取认证网址： 解决方法：在浏览器地址栏输入认证网关地址即可； 6.可以获取到认证界面却无法发送和收到验证码 解决方法：一般表现为网页显示不正常，此时需更换浏览器，建议使用谷歌浏览器或360急速模式，认证页面设配性差问题已经反馈集团；

    1.2 有线网络故障¶

    1.区域性有线网络故障 解决方法：一般为网络环接，查找该交换机下所有网络设备接口是否出现环接； 2.个别电脑网络时有时无 解决方法：查看电脑网络连接图标如为感叹号说明IP冲突更换IP，如为红叉网线连接不良，检测网线或设备接口； 3.网络连接显示正常只有个别网页或应用可以使用 解决方法：一般为DNS解析出现故障，可在电脑设置里配置备用DNS服务器为114.114.114.114，询问运营商是否更换DNS；

    1.3 AP日常检查¶

    H3C AC控制器：

    通过Console口来连接AC控制器，使用超级终端进行快速连接，注意波特率设为9600 查看终端在线状态dis wlan ap all 查看终端工作状态dis wlan ap all verbose

    2.程控交换系统¶

    2.1 电话有杂音¶

    解决方法：如为客房一般为卫生间免提电话模块故障，将免提电话模块拆卸后，查看杂音是否消失；其他原因产生的杂音检查电话插口面板接线，一般为接线不良或者有破皮修复即可；其次测试话机相应区域的大对数模块，查看是否正常，最后查看服务器机房配线架；

    2.2 电话无声音¶

    解决方法：换一部话机和连接线测试是否正常，首先排除话机和连接的故障，其次测试话机相应区域的大对数模块，查看是否正常，最后查看服务器机房配线架；

    3.电视机使用¶

    3.1遥控器使用说明¶

    一、遥控器平面图说明 以上为遥控器平面图，遥控器操作按键分为两部分，红色框内为电视机操作按键，蓝色框内为机顶盒操作按键。 二、正常开机流程 客房插卡后电视机及机顶盒会自动开机，开机后出现×××欢迎界面点击蓝色框内“OK”键，即可观看广播电视，如未自动开机，请按电视机开机键开机。

    3.2常见问题解决方法¶

    1. 电视显示 “无信号” 解决方法：1.检查机顶盒有无开机，如未开机请点击机顶盒开机键；2.点击遥控器“信号源”按键，出现信号源选择界面，选中“HDML1”，点击红色框内的“OK”键，即可解决。如按以上两个步骤操作后无解决问题联系工程部弱电组报修。
    2. 电视机开机后显示二维码界面 解决方法：按下“信号源”按键，出现信号源选择界面，选中“HDML1”，点击红色框内的“OK”键，即可解决。
    3. 电视机显示“无识别智能卡” 解决方法：请取出机顶盒智能卡，使用橡皮或棉质物品对芯片一侧擦拭数次，插入机顶盒即可解决，如未解决请联系工程部弱电组报修。
    4. 遥控器按键无反应，指“蓝色区域”按键 解决方法：①拔下机顶盒上的黑色接收器，重新插入接收器，此时接收器闪一下×××说明遥控器和机顶盒未配对或配对丢失，需重新配对，②同时按住机顶盒开机键和蓝色区域OK键3秒，接收器会再次闪×××配对成功，此时可正常使用遥控器。 注意：①②步骤需连续进行，操作完①步骤间隔不超过5秒操作步骤②；
    5. 机顶盒开机灯显示正常，HDML线连接正常，信号源选择正确，电视依旧无信号，可一次更换HDML接口、HDML线材、机顶盒进行故障排除。

    7.PMS管理系统¶

    7.1皓谷系统¶

    皓谷系统为开元酒店管理皓谷科技公司旗下PMS管理系统，现基本开元酒店基本转向此系统，系统流程操作性一般但是使用较为简洁可以满足酒店一般使用。此系统使用跨平台开发语言，安装时需要先装运行环境 Adobe AIR类似Java虚拟机，才可安装iHotel客户端，安装文件小、安装速度快，各系统为独立模块按需安装，提供官方接口服务内容丰富，但使用流畅度一般因运行机制原因其在（HTML+Flash下进行）。

    7.1.1皓谷前台系统日常维护¶

    对于酒店IT维护人员上手没有难度，刚接触时多上手操作。半小时入手。表单（前台账单、餐饮账单、客人清单等）维护稍有难度，皓谷和西软一样，表单本来就是不属于酒店IT维护，如无能力请联系售后，但是笔者提倡研究，但研究前必须对账单格式数据进行备份，防止带来不必要的麻烦，皓谷的表单会进行云同步，所以修改后本酒店所有账单格式发生变化（在操作账单格式更新的情况下），西软账单修改完成只更新本机账单格式，其他电脑如需更新可复制其账单格式到西软安装目录下进行替换.

    总结：笔者个人喜欢使用西软，但西软对酒店IT要求较高，其全家桶安装方式笔者也觉得很好一次安装走到哪都可随时进行维护，本地化部署不受环境影响（也有云同步此需求根据酒店运营而选择）；皓谷使用云部署目前流行趋势，但受实际环境影响如网络波动、云服务器性能等原因其稳定性有待提高，另外皓谷权限警告提醒不如西软人性化，给维护带来不便，如何选用仁者见仁，笔者在此不对两者做出评判，各有各的优点，当然西软的维护费用和品质一样高。

    8.2门锁故障恢复¶

    门锁恢复刷卡流程：总裁卡→管理卡→总控卡→领班卡→楼层卡→房号卡→时钟卡 1) 设置总裁卡(将总裁卡靠近门锁感应头，蜂鸣器“嘀”一声响，绿灯亮一下，表示总裁卡设置成功;) 2) 设置管理卡(将管理卡靠近门锁感应头，蜂鸣器“嘀”一声响，绿灯亮；移开卡后绿灯长亮闪烁，表示管理卡设置成功;) 3) 设置领班卡、清洁卡、和楼层卡、房号卡(设置完管理卡后，在绿灯长亮，红灯闪烁期间，进行依次设置领班卡、清洁卡、楼层卡和房号卡；如果指示灯灭了，这需重新管理卡进入设置状态;) 4) 设置时钟卡(校对门锁时钟，需要门锁的已退出设置状态，即所有的灯灭后再进行时钟校对;) 5) 完成设置
